The BiplotGUI package homepage

The BiplotGUI package for R makes it easy to construct and interact with biplots.

Biplots

Biplots can be interpreted as graphs in which observations are represented as points and variables are represented as calibrated biplot axes. Such representations make it easy to visualise multivariate data in two or three dimensions. The biplots of the BiplotGUI package are based on the book by Gower and Hand (1996).

Requirements

  • At present, BiplotGUI runs only in Windows.
  • BiplotGUI requires at least version 2.7 of R. Version 2.7.1 can be downloaded from CRAN.
  • Various other R packages are required:
    • MASS (included in the base distribution of R)
    • tcltk (included in the base distribution of R)
    • tcltk2 (at least version 1.0-7)
    • tkrplot (at least version 0.0-18)
  • While not required, some R packages are highly recommended. Without them, certain features of the BiplotGUI package will be unavailable, while the other features will operate normally.
    • KernSmooth for the estimation of point densities
    • rgl for the creation of dynamic 3D biplots
    • vcd for the colours to be used in the representation of point densities
    • xlsReadWrite for the import of data from Excel 1997-2003 files

Resources

The features of the BiplotGUI package are documented in a manual  which is also included in the package, and which can be accessed via the Help menu of the GUI.
